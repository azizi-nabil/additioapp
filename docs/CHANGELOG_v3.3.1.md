# Changelog v3.3.1

## UI/UX Improvements
- **Class Tabs**: Added text labels (Students, Attendance, Grades, Behavior) under the icons in the Class Detail page for better navigation.
- **Tab Layout**: Optimized tab height and text size (10sp) for better visibility.
- **Hidden Features**: Added a tip in Settings for clearing the replacement date (Long-press).

## Fixes & Polish
- Removed redundant tab title chip from the toolbar.
- Standardized tab appearance to match Planner page style.
- Added "Clear Replacement Date" functionality to revert absence status to Pending.
